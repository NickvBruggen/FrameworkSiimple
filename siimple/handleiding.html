Je kan via de site van Siimple naar Git toegaan, daar kan je een bestand downloaden. In dat bestand staan andere bestanden zoals de css bestanden. Als je het gedownload hebt kan het openen in je editor. Als je in je editor zet kan je de css bestanden linken aan je html files. Dit kan met deze code: Als je dit gedaan hebt dan kan je op de site van Siimple kijken naar welke codes je kan gebruiken en welke kleuren dus. De opties die je kan gebruiken vallen mee maar dan heb je wel een responsive site.
